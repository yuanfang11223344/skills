# canvas

来源: https://github.com/markdown-viewer/skills

## 使用
安装: npx skills add markdown-viewer/skills

此 skill 生成 Markdown 原生内嵌图表，无需外部图片链接，在 Obsidian 等 Markdown 阅读器中直接渲染。

## 安装说明
由于 GitHub SSL 连接问题，此为基础版本。建议通过 npm 获取完整 skill：
