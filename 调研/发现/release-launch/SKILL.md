# Release Launch

Use this skill after the arXiv submission is ready.

## Workflow

1. Inventory launch assets: arXiv link, project page, code, data, demo, video,
   model weights, slides, and citation.
2. Draft launch copy for X/Twitter, LinkedIn, lab website, email, and README.
3. Schedule coauthor approval and link update sequence.
4. Include limitations and responsible-use notes where needed.

## Output

Return `Launch asset checklist`, `Announcement copy`, `Link update plan`,
`Coauthor tasks`, and `Post-launch monitoring`.
